### 📊 [Kenya Mobile Credit Risk Profiler](https://github.com/thebrianmwenda/kenya-credit-risk-profiler)
> *Segmenting mobile borrower risk using behavioral and financial survey data*

An end-to-end data analytics project analyzing mobile loan adoption and credit risk patterns among **286 Nairobi CBD respondents**. Applies machine learning to classify borrowers into risk segments — directly relevant to every fintech and bank operating in Kenya.

**Pipeline:**
```
Raw Survey Data (n=286)
    → Data Cleaning & EDA (Pandas)
    → Borrower Segmentation (K-Means Clustering)
    → Risk Classification (Logistic Regression + Decision Tree)
    → Visualization Dashboard (Seaborn + Matplotlib)
    → Stata Validation (Multiple Linear Regression)
```

**Frameworks applied:** Technology Acceptance Model (TAM) · Theory of Planned Behavior · Behavioral Finance Theory
